# 🧵 Cross Stitch App

Welcome to the **Cross Stitch App**! 🎨 This interactive web application allows you to create and customize cross-stitch patterns digitally. Whether you're a seasoned stitcher or just exploring the craft, this tool helps you design grids, choose colors, and save your creations.

## ✨ Features

- 📏 **Customizable Grid**: Set your desired height and width (up to 20x20) to create the perfect canvas.
- 🎨 **Color Picker**: Select any color for your stitches with an intuitive color input.
- 🖼️ **Save Your Work**: Export your cross-stitch pattern as an image using the save button.
- 🧹 **Clear Canvas**: Start fresh anytime with the clear button.
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices.

## 🚀 How to Use

1. Adjust the **Height** and **Width** inputs to define your grid size.
2. Pick a **Color** for your stitches.
3. Click **Create Grid** to generate your canvas.
4. Click on grid cells to fill them with your selected color.
5. Use **Save Image** to download your pattern.
6. **Clear** the grid to start over.
